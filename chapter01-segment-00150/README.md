# Problem

* READ A VALUE FROM IN.X AND WRITE THE VALUE TO OUT.X
* READ A VALUE FROM IN.A AND WRITE THE VALUE TO OUT.A

# Solution

83 CYCLES / 8 NODES / 8 INSTR

```

@1

@2
```
